# Payload Generator Skill

## Objective
Generate and mutate payloads for various vulnerability types.

## Instructions
1. Understand the injection context
2. Generate appropriate payloads for the vulnerability type
3. Apply encoding/obfuscation as needed
4. Consider WAF bypass techniques

## Payload Types
- SQL Injection payloads
- XSS payloads
- Command injection payloads
- SSTI payloads
- Path traversal payloads

## MCP Tools to use
- url_encode / url_decode: Encode payloads
- base64_encode / base64_decode: Base64 encoding
