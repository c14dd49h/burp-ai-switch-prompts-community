# Analyze Request Skill

## Objective
Perform deep analysis of HTTP requests and responses to identify potential security issues.

## Instructions
1. Parse all request parameters (query, body, headers, cookies)
2. Identify authentication mechanisms
3. Look for sensitive data in requests/responses
4. Check for security headers
5. Identify potential injection points

## MCP Tools to use
- params_extract: Extract all parameters
- request_parse: Parse request structure
- response_parse: Parse response structure
- find_reflected: Check for input reflections

## Keywords
analyze request, analyse, request analysis
