# Fingerprint Skill

## Objective
Identify technologies, frameworks, and server configurations used by the target application.

## Instructions
1. Analyze HTTP headers for server information
2. Check response bodies for technology fingerprints
3. Identify JavaScript frameworks and libraries
4. Detect CMS and backend technologies

## MCP Tools to use
- http1_request: Send probing requests
- proxy_http_history: Analyze existing traffic
- site_map: Review discovered content
